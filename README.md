## About Yummi

Yummi is a pizza store application built in the Laravel framework.

Other technologies used, including Laravel, can be found in the following list:

- [Laravel official website](https://laravel.com/)
- [PhpMyAdmin](https://www.phpmyadmin.net/)
- [XAMPP](https://www.apachefriends.org/index.html)
- Written in [PhpStorm](https://www.jetbrains.com/phpstorm/)
- [Bootstrap](https://getbootstrap.com/)
- [Apache](https://httpd.apache.org/)
- [MySQL](https://www.mysql.com/)
- [PHP](https://www.php.net/) (ofcourse)

All of this gives us the infamous [WAMP](https://en.wikipedia.org/wiki/WampServer) stack.

## Functionality

The functionality this web application provides is very simple, yet very effective.

As a logged in user: 
- you are able to order pizza's from the PIZZA MENU which are then added to the cart
- you are able to review your pizza cart, and change the quantity of the order if you desire so
- you are able to proceed to checkout your order, where you can enter your details for delivery/billing

## Deployment

The application is intended to be deployed on the [Heroku](https://www.heroku.com/) cloud application platform, alongside the database being uploaded to the free [Remote MySQL](https://remotemysql.com/) service.
